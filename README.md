# Number Analyzer

## 🇺🇸 English

### 📌 Description

This is a simple Python project that analyzes numbers within a given interval. The program separates even and odd numbers, calculates their sums, counts, and averages. It is also an *odd* (a playful pun on “ode”) to mathematics — a small tribute to the beauty of numbers.

### 🚀 Features

* Lists all even numbers
* Lists all odd numbers
* Counts total even and odd numbers
* Calculates sums
* Calculates averages

### 🛠️ Technologies

* Python

### ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/gabart2004/number-analyzer.git

2. Navigate to the folder:
   cd number-analyzer

3. Run the program:
   python main.py
